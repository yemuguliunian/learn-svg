## stroke

定义了给定图形元素的外轮廓的颜色。它的默认值是 none。

### 示例

用 stroke 属性画一条绿色的直线.

```html
<svg height="50" width="300">
    <path stroke="green" d="M5 20 1215 0" />
</svg>
```

<div class="demo">
    <svg height="50" width="300">
        <path stroke="green" d="M5 20 1215 0" />
    </svg>
</div>

用 Stroke 属性画一个拥有蓝色边框的黑色的圆形

```html
<svg height="200" width="200">
    <circle cx="50" cy="50" r="40" stroke="blue" stroke-width="2" fill="black" />
</svg>
```

### 元素

下列元素可以使用 `stroke` 属性：

-   形状元素
-   文本内容元素
